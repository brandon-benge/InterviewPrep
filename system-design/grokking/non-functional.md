# Non-Functional Requirements (NFRs)

## 1. Scale
- Daily Active Users / Peak RPS / Read:Write Ratio
- Storage Size / Growth Rate
- Geographic Scope

## 2. Availability & Reliability
- SLA / RTO / RPO
- Durability Requirements
- Failure Handling / DR Strategy

## 3. Performance
- p95 Latency
- Throughput Targets
- Async vs Sync / Real-Time Needs

## 4. Consistency & Distributed Tradeoffs
- Strong vs Eventual Consistency
- Idempotency / Ordering Guarantees
- CAP Tradeoff (CP vs AP)
- PACELC (Latency vs Consistency)
- Partitioning / Replication Strategy

## 5. Security & Compliance
- AuthN/AuthZ / Encryption
- Auditability / Tenant Isolation
- Compliance Constraints

## 6. Observability & Operations
- Metrics / Logging / Tracing
- Alerting / SLOs
- Deployment Strategy

## 7. Cost & Scalability
- Horizontal Scaling / Autoscaling
- Retention Policy
- Cost Sensitivities
